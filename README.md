# BCI Mind Wandering

Mind wandering is hard to study since participants are not aware they are mind wandering as it is happening. The ROAMM dataset (Sun et al., 2025) includes multi-modal (EEG, eye tracking) data on mind wandering participants while reading. The data can be obtained from https://openneuro.org/datasets/ds007629/versions/1.1.0.

<i>This material was initially created as part of a group project in the Brain Computer Interfacing course at Tilburg University.</i>
